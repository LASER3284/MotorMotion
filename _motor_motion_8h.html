Detailed Description

This file contains the mostly virtual class MotorMotion, which is implemented by other classes.

This allows for a common mode of control between different motor controllers using a similar API. The class in this file is purely virtual and cannot be instantiated; it is implemented through TalonFXMotion and SparkMaxMotion.
